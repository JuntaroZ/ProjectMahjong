## 実行ファイルの概要

`ProjectMahjong.exe` は、設定された手牌を画像で表示し、手牌を変更しながらシャンテン、和了候補、成立役、点数を確認するためのアプリケーションです。

主な機能は以下の通りです。

- 手牌14枚を画像で表示します。
- 一番右の牌をツモ牌またはロン牌として扱います。
- テンパイ時と和了時に、成立役を1役1行で表示します。
- 成立役には翻数と点数を表示します。
- テンパイ時は和了候補牌を2行目左側に表示します。
- 牌を変更すると、シャンテン、成立役、点数、和了候補を再判定します。
- 同じ牌が5枚以上になる状態は不正として扱い、エラー表示します。
- 配牌ソートボタンで、手牌を萬子、筒子、索子、字牌の順に並べ替えできます。
- ツモ／ロン切替ボタンで、14枚目の扱いをツモまたはロンに切り替えできます。

## 操作方法

| 操作 | 内容 |
| --- | --- |
| 左キー | 選択中の牌を左へ移動します。左端からさらに左へ移動すると右端へ移動します。 |
| 右キー | 選択中の牌を右へ移動します。右端からさらに右へ移動すると左端へ移動します。 |
| 上キー | 選択中の牌を次の牌へ変更します。例: `m9` の次は `p1`、`z7` の次は `m1` です。 |
| 下キー | 選択中の牌を前の牌へ変更します。 |
| Shift + 上キー | 図柄を次へ変更します。萬子 → 筒子 → 索子 → 字牌 → 萬子の順で切り替わり、番号は1になります。 |
| Shift + 下キー | 図柄を前へ変更します。番号は1になります。 |
| 配牌ソートボタン | 14枚ある場合は右端のツモ／ロン牌を残し、それ以外の手牌を萬子、筒子、索子、字牌の順にソートします。 |
| ツモ／ロン切替ボタン | 14枚目の扱いをツモとロンで切り替えます。 |

スペースキーによるツモ／ロン切り替えは現在使用していません。

## 表示について

- 黄色い枠は現在選択中の牌です。
- 一番右の牌はツモ牌またはロン牌です。
- 一番右の牌の上には「ツモ牌」「ツモ」「ロン」のいずれかが表示されます。
- 画面上部には「和了」「テンパイ」「イーシャンテン」「リャンシャンテン」を表示します。
- 成立役はテンパイ時と和了時だけ表示します。
- 成立役は1役1行で、`役名（翻数 / 点数）` の形式で表示します。
- 和了候補がある場合、2行目の左側に「和了候補」と候補牌が表示されます。
- 同じ牌が5枚以上になる場合、選択中の牌がグレー表示になり、上部に「4個以上は選べません」と表示されます。この状態では左右移動できません。

## 点数表示について

- 表示している点数は、役ごとの表示用点数です。
- 親、子、ロン、ツモなどの支払い表記は表示しません。
- 符計算は現在、表示用の簡易計算です。
  - 七対子: 25符
  - 平和ツモ: 20符
  - その他: 30符
- 満貫以上は、満貫、跳満、倍満、三倍満、役満の点数として表示します。

## ソース構成

- `ProjectMahjong.cpp`
  - 画面表示
  - キーボード操作
  - ボタンクリック処理
  - 牌画像の読み込み
- `MahjongYaku.h` / `MahjongYaku.cpp`
  - 役判定の公開API
  - シャンテン計算
  - 和了候補探索
  - 表示用役一覧と点数計算
- `MahjongHandAnalysis.h` / `MahjongHandAnalysis.cpp`
  - 手牌の分解
  - 面子、雀頭、七対子、国士無双などの形判定
- `YakuRules`
  - 各役の判定ルール
  - 役名、翻数、役満判定
